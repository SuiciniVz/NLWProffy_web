# Getting Started

```bash
# Clone Repository
$ git clone https://github.com/SuiciniVz/NLWProffy_web.git

# Install Dependencies
$ yarn install

# Run Aplication
$ yarn start
```
:eyes: Go to http://localhost:3000/ to see the result.
